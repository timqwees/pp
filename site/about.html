О проекте

Разработка PHP фреймворка с компонентным подходом, вдохновленного React/JSX.

Посмотреть пример работы фреймворка

Цели и задачи

Основные цели

  • ✓ Создание легковесного PHP фреймворка
  • ✓ Реализация компонентного подхода
  • ✓ Разработка системы управления состоянием
  • ✓ Создание виртуального DOM

Задачи проекта

  • ✓ Исследование существующих решений
  • ✓ Проектирование архитектуры
  • ✓ Реализация базового функционала
  • ✓ Написание документации

Пример реализации

Todo List

Для демонстрации работы фреймворка создан пример списка задач, который показывает основные возможности:

  • Компонентный подход
  • Управление состоянием
  • Рендеринг HTML
  • Взаимодействие с API

Как запустить пример:

  1. Установите зависимости: composer install
  2. Запустите PHP-сервер: php -S localhost:8000 -t examples
  3. Откройте в браузере: http://localhost:8000/todo.php
