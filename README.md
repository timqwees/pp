# Проектная практика

## Описание проекта
Этот репозиторий содержит материалы по проектной (учебной) практике.

## Структура проекта
- `docs/` - Документация проекта в формате Markdown
- `reports/` - Отчёты по практике
- `website/` - Статический веб-сайт проекта

## Требования
- Git
- Hugo (опционально, для генерации статического сайта)
- Markdown редактор

## Установка и запуск
1. Клонируйте репозиторий:
```bash
git clone [URL репозитория]
```

2. Для работы с документацией:
- Откройте файлы в `docs/` в любом Markdown редакторе

3. Для работы с веб-сайтом:
- Перейдите в директорию `website/`
- Следуйте инструкциям в README.md внутри директории website
