# PHP Framework

## Описание проекта
PHP Framework - это современный фреймворк с компонентной архитектурой, вдохновленной React. Фреймворк предоставляет удобный интерфейс для создания веб-приложений с использованием компонентного подхода.

## Основные возможности

### Компоненты
- **Auth** - компонент для аутентификации и авторизации
- **Post** - компонент для работы с постами и комментариями
- **Diary** - компонент для ведения личного дневника

### API Endpoints
- `/api/posts` - управление постами (GET, POST)
- `/api/posts/comment` - добавление комментариев (POST)
- `/api/diary/entries` - управление записями дневника (GET, POST)

### Демо-интерфейс
- Страница компонентов с примерами использования
- Интерфейс для тестирования API
- Демонстрационная панель управления

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/php-framework.git
cd php-framework
```

2. Установите зависимости:
```bash
composer install
```

3. Запустите демо-сервер:
```bash
php -S localhost:8000 -t examples
```

4. Откройте в браузере:
- Демо: http://localhost:8000/demo.php
- Компоненты: http://localhost:8000/components.php
- API тестирование: http://localhost:8000/api_test.php

## Структура проекта

```
├── src/                    # Исходный код фреймворка
│   ├── Components/        # Компоненты
│   │   ├── Auth/         # Компонент аутентификации
│   │   ├── Post/         # Компонент постов
│   │   └── Diary/        # Компонент дневника
│   └── Core/             # Ядро фреймворка
├── examples/              # Примеры использования
│   ├── templates/        # Шаблоны страниц
│   ├── components.php    # Демо компонентов
│   ├── api_test.php      # Тестирование API
│   └── demo.php          # Главная демо-страница
└── docs/                 # Документация
```
