# Отчёты по проектной практике

В этой директории будут храниться все отчёты по проектной практике в формате PDF.

## Содержание

1. Итоговый отчёт (будет добавлен после завершения практики)
2. Промежуточные отчёты (если потребуются)

## Формат отчётов

Все отчёты должны быть представлены в формате PDF и содержать:
- Титульный лист
- Содержание
- Основной текст
- Список использованных источников
- Приложения (если есть)
