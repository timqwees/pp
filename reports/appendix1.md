# Структура проекта и описание файлов

## Структура проекта

```
├── src/                    # Исходный код фреймворка
│   ├── Components/        # Компоненты
│   │   ├── Auth/         # Компонент аутентификации
│   │   │   └── Auth.php  # Основной класс компонента
│   │   ├── Post/         # Компонент постов
│   │   │   └── Post.php  # Основной класс компонента
│   │   └── Diary/        # Компонент дневника
│   │       └── Diary.php # Основной класс компонента
│   └── Core/             # Ядро фреймворка
│       └── Component.php # Базовый класс компонента
├── examples/              # Примеры использования
│   ├── templates/        # Шаблоны страниц
│   │   ├── dashboard.php # Шаблон панели управления
│   │   └── layout.php    # Базовый шаблон
│   ├── components.php    # Демо компонентов
│   ├── api_test.php      # Тестирование API
│   └── demo.php          # Главная демо-страница
├── docs/                 # Документация
│   └── README.md         # Основная документация
└── reports/             # Отчеты
    └── final_report.md   # Итоговый отчет
```

## Описание файлов

### Исходный код (src/)

#### Components/Auth/Auth.php
Основной класс компонента аутентификации. Реализует:
- Управление сессиями пользователей
- Методы авторизации и аутентификации
- Проверку прав доступа

#### Components/Post/Post.php
Компонент для работы с постами. Включает:
- Создание и редактирование постов
- Управление комментариями
- Форматирование контента

#### Components/Diary/Diary.php
Компонент дневника. Предоставляет:
- Создание и редактирование записей
- Управление форматированием
- Фильтрацию и поиск

#### Core/Component.php
Базовый класс для всех компонентов. Содержит:
- Общие методы жизненного цикла
- Систему событий
- Управление состоянием

### Примеры (examples/)

#### templates/dashboard.php
Шаблон панели управления с:
- Навигационным меню
- Областью контента
- Модальными окнами

#### templates/layout.php
Базовый шаблон, включающий:
- Общую структуру страницы
- Подключение стилей и скриптов
- Общие элементы интерфейса

#### components.php
Демонстрационная страница компонентов:
- Примеры использования Auth
- Демо Post компонента
- Тестирование Diary

#### api_test.php
Интерфейс для тестирования API:
- Формы для отправки запросов
- Отображение ответов
- Документация endpoints

#### demo.php
Главная демо-страница:
- Обзор возможностей
- Навигация по разделам
- Примеры использования

### Документация (docs/)

#### README.md
Основная документация проекта:
- Описание компонентов
- Руководство по API
- Примеры использования
- Инструкции по установке

Итоговый отчет по проектной практике:
