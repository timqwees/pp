# Задание по проектной практике

## Вариативная часть

В рамках проектной практики необходимо выбрать одну из технологий из репозитория [codecrafters-io/build-your-own-x](https://github.com/codecrafters-io/build-your-own-x) и выполнить следующие задачи:

1. Исследовать принципы работы выбранной технологии
2. Создать собственную реализацию с нуля
3. Подготовить подробное техническое руководство
4. Внести модификации и улучшения в проект

## Требования к проекту

### Документация
- Подробное описание процесса разработки
- Техническое руководство для начинающих
- Диаграммы и схемы архитектуры
- Описание модификаций и улучшений

### Код
- Чистый, хорошо документированный код
- Тесты для проверки функциональности
- Примеры использования
- Инструкции по установке и запуску

### Сайт
- Статический сайт с описанием проекта
- Документация в формате веб-страниц
- Информация об участниках
- Журнал прогресса

## Критерии оценки

1. Качество реализации
2. Полнота документации
3. Оригинальность модификаций
4. Презентация результатов
